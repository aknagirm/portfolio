Time Complexity: The time complexity of an algorithm quantifies the amount of time taken by an algorithm to run as a function of the length of the input.

It needs to be noted that this calculation is asymptotically efficient. In real world execution time can differ based on the input numbers. There can be best case (least time needed) scenario, worst case (maximum time needed) scenarion and average scenario performance check.

We denote best case, worst casea and average with big Omega (Ω), big O (O) and big Theta (Θ) notation respectively
